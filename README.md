# Calculator

## Introduction

实现了简单的整数公式的四则运算

## Install

```
npm install zzy_calculator --save
```

## How to use

```js
const CalculateFormula = require("zzy_calculator")
```

```js
const expression = "a+b"
const variables = {
    a : 6, b : -5
}

let result = CalculateFormula(expression, variables)
```
